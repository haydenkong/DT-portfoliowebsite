Skills

HTML

HTML is a markup language used for creating web pages. It provides the structure and content of a webpage, including headings, paragraphs, images, links, and more.

Level: Efficient

CSS

CSS is a styling language used for describing the look and formatting of a document written in HTML. It provides the visual appearance of a webpage, including colors, layout, fonts, and more.

Level: Intermediate

Javascript

JavaScript (JS) allows you to add interactivity and dynamic behavior to your web pages. It is commonly used for creating interactive features, validating forms, manipulating HTML elements, and much more.

Level: Beginner

VueJS

Vue.js is a progressive JavaScript framework used for building user interfaces. It allows you to create reusable components and build interactive web applications with ease.

Level: Learning

Python

Python is a high-level programming language known for its simplicity and readability. It is widely used for web development, data analysis, artificial intelligence, and more. Python's extensive library ecosystem and easy syntax make it a popular choice among developers.

Level: Intermediate
